# Protocol 33: Aesthetic Code

## 1. Global CSS Variables (Dark Mode Palette)
The color system is rooted in Period 9 Feng Shui principles (Li Trigram / Fire within Metal).

- **The Void (Background):** `#0d0514` (A deep midnight/indigo twilight purple, representing infinite space, rest, and pure potential).
- **Alchemical Gold (Accent):** `#ddb940` (Used for primary accents, calls-to-action, active states, and Illuminating structural lines. Communicates success and Sovereign affluence).
- **Platinum (Structure):** `#e5e7eb` or `rgba(255, 255, 255, 0.15)` (Used for borders to represent the "Metal" element's clinical precision).
- **Text (Primary):** `#f3f4f6` (Off-white for readability against The Void).
- **Text (Muted):** `#9ca3af` (For secondary information).

## 2. Typography
- **Primary Font:** `Space Grotesk`
  - Used for all headings, body text, and UI elements to provide a modern, structural, and slightly geometric feel that bridges ancient wisdom with technology.

## 3. Design Rules
- **"Structure is Freedom":** 
  - Strict, clean geometry anchors fluid energy. 
  - Use `1px solid` platinum/subtle borders aggressively to define containers, sections, and cards.
  - Generous padding (e.g., `p-6`, `p-8`) inside containers to allow elements to breathe.
  - High contrast between The Void background and Alchemical Gold accents.
  - Minimalistic but highly structured layouts.
